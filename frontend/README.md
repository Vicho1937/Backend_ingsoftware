# Ruta Local - Frontend

Frontend de la aplicación Ruta Local construido con React + Vite.

## Tecnologías

- React 18
- React Router DOM
- Axios
- Leaflet (Mapas)
- Vite

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en http://localhost:5173

## Build

```bash
npm run build
```

## Características

- ✅ Autenticación con JWT
- ✅ Exploración de rutas locales
- ✅ Sistema de favoritos
- ✅ Reseñas y calificaciones
- ✅ Chatbot con IA
- ✅ Visualización en mapa
- ✅ Filtros y búsqueda
- ✅ Perfil de usuario
